    <h2 class="ai-accent">01 :: INTELLIGENCE_TYPES</h2>
    <p>Memorize these three domains for the exam:</p>
    <ul>
      <li><strong>Data Science (DS):</strong> Input is data (Numbers/Stats). <em>Ex: Price Prediction.</em></li>
      <li><strong>Computer Vision (CV):</strong> Input is images/pixels. <em>Ex: Face Lock.</em></li>
      <li><strong>Natural Language Processing (NLP):</strong> Input is text/voice. <em>Ex: Alexa, ChatGPT.</em></li>
    </ul>
  </div>

  <div class="revision-card border-ai">
    <h2 class="ai-accent">02 :: AI_PROJECT_CYCLE</h2>
    <p>The 5 stages you must write in order:</p>
    <ol>
      <li><strong>Problem Scoping:</strong> Defining the goal (4Ws Canvas).</li>
      <li><strong>Data Acquisition:</strong> Collecting reliable data.</li>
      <li><strong>Data Exploration:</strong> Visualizing data to find patterns.</li>
      <li><strong>Modelling:</strong> Creating the algorithm (Rule-based vs Learning-based).</li>
      <li><strong>Evaluation:</strong> Testing accuracy.</li>
    </ol>
    
    <div class="exam-alert">
      ⚠️ <strong>EXAM TIP:</strong> Don't confuse "Acquisition" (Collecting) with "Exploration" (Analyzing).
    </div>
  </div>

  <div class="revision-card border-ai">
    <h2 class="ai-accent">03 :: NEURAL_NETWORKS</h2>
    <p>Neural Networks mimic the human brain. They consist of three layers:</p>
    <ul>
      <li><strong>Input Layer:</strong> Receives data.</li>
      <li><strong>Hidden Layer(s):</strong> Processes data (The "Black Box").</li>
      <li><strong>Output Layer:</strong> Gives the final result.</li>
    </ul>
